Do-It-Yourself Photshop: Minecraft House

Using the creative coding software, p5.js, I developed a Do-It-Yourself photoshop program. Create your own house using items within the video game, Minecraft!


  '1' = Dirt/Grass Block
  '2' = Spruce Plank
  '3' = Birch Plank
  '4' = Spruce Log
  '5' = Bed
  '6' = Doors
  '7' = Torch
  '8' = Sugarcane
  '9' = Crafting Table
  '0' = Furnace
  'c' or 'C' = Cobbelstone Block
  'x' or 'X' = Clear Canvas
  'p' or 'P' = Save Artwork
